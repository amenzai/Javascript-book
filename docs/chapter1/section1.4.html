字符串

是个特殊的数组

var str = 'hello'
str[0] // h

想要使用数组方法需要借助call方法。

var s = 'hello';

s.join(' ') // TypeError: s.join is not a function

Array.prototype.join.call(s, ' ') // "h e l l o"
