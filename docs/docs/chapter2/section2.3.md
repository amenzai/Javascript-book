# Boolean 对象

产生布尔值的场景：

1. if判断
2. Boolean()数据类型转换
3. !!query（二次取非）

## Boolean()

类型转换作用
```js
Boolean(undefined) // false
Boolean(null) // false
Boolean(0) // false
Boolean('') // false
Boolean(NaN) // false

Boolean(1) // true
Boolean('false') // true
Boolean([]) // true
Boolean({}) // true
Boolean(function () {}) // true
Boolean(/foo/) // true
```
